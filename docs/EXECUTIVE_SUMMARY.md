# Executive Summary — LLM Token Cost Audit

A production support agent handling 1M requests/month was spending $21,612/month on LLM API costs; this audit reduced it to $3,585/month, an 83% reduction worth ~$216K annually. The waste came from static overhead: a 4,784-token system prompt and tool schema block re-sent with every single request, verbose mandated outputs, and running every request on a mid-tier model regardless of complexity. Five levers fixed it -- prompt compression, tool schema trimming, prompt caching, model routing, and output discipline -- with zero policy content removed. Savings scale linearly with volume and the framework is reusable against any agent config. Assumptions (routing mix, cache hit rate) are stated explicitly and should be validated against production transcripts before committing the number.
